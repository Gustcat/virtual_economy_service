# virtual_economy_service

## Описание проекта
Сервис управления виртуальной экономикой для приложения с микроплатежами и премиум-контентом.
Предоставляет функционал управления виртуальными товарами, балансом пользователей и транзакциями.  
Основные возможности:

- Покупка виртуальных товаров (`consumable` и `permanent`)
- Пополнение баланса пользователя с поддержкой идемпотентности
- Получение и кэширование инвентаря пользователя
- Использование consumable товаров
- Аналитика популярных товаров
- Фоновая задача ежедневной очистки кеша инвентаря через Celery

## Развертывание

Клонировать репозиторий
```
git clone https://github.com/Gustcat/virtual_economy_service.git
cd virtual_economy_service
```
Перед запуском необходимо создать файл .env на основе .env.example и указать необходимые переменные окружения.
В корневой директории проекта запустите сборку сети контейнеров:
```
docker compose up -d
```
Перейти внутрь контейнера бэкенда:
```
docker-compose exec backend bash
```
Применить существующие миграции:
```
alembic upgrade head
```
